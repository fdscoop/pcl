```
Connection: ws://supabase-realtime.example.com

{
  "type": "subscribe",
  "id": 1,
```

## Summary

The dashboard notifications system is a complete, real-time notification delivery system that:

1. **Fetches** notifications from Supabase
2. **Subscribes** to real-time changes via WebSocket
3. **Displays** notifications in a professional UI
4. **Allows** users to mark as read
5. **Updates** automatically when changes occur
6. **Handles** errors gracefully

All integrated into the club-owner main dashboard with zero performance impact and full TypeScript type safety.
